
# Sistema de PDV e Gestão de Estoque

Sistema completo de frente de caixa e controle de estoque desenvolvido para otimizar operações de varejo (Adega/Mercado). O projeto foca em agilidade e precisão no gerenciamento de produtos.

## 🛠️ Tecnologias e Arquitetura
* **Frontend:** HTML5, CSS3 e JavaScript (ES6+).
* **Backend & Database:** Firebase Realtime Database (Sincronização em tempo real).
* **Hospedagem/Cloud:** Deploy em infraestrutura com IP personalizado.
* **Ferramentas:** Desenvolvido utilizando Cursor AI e Git.

## ✨ Funcionalidades
* **Venda Rápida (PDV):** Interface otimizada para fluxo de caixa.
* **Estoque:** Entrada e saída de produtos com atualização instantânea.
* **Responsividade:** Design adaptável para desktop e dispositivos móveis.
